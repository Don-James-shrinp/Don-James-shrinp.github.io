变量的作用域

  • 局部变量的作用域仅限于定义它的方法
  • 成员变量的作用域在整个类内部都是可见的
  • Java会给成员变量一个初始值
  • Java不会给局部变量赋予初始值
  • 在同一个类中，局部变量和成员变量重名时，局部变量具有更高的优先级

包名

包名以小写字母组成，不能以’.’开头或者结尾。

类的访问控制

  • 类的访问控制只有两种，public和无修饰符

  • public：同一包中的类可访问，不同包中的类也可访问（通过import）。

  • 无修饰符：同一包中的类可访问，不同包中的类不能访问（就算import了也无法访问）。

类成员的访问控制

  • public：可以被其他任何对象访问（前提是对类成员所在的类有访问权限）

  • protected：只能被同一类及其子类的实例对象访问

  • private：只能被这个类本身访问，类外不可见

  • default：仅允许同一个包内的访问，被称为“包访问权限”

  • 默认为default，即没有修饰符的情况

对象之间的赋值

java对象之间的赋值，实际上是对另一个对象的引用，而不是拷贝。

因此函数传参，传入对象时相当于直接对原对象进行操作
